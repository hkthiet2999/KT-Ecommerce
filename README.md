### My project in the research and development process ...
- Front-end: ReactJS
- Database: MongoDB
- Back-end: NodeJS

### Usage
1. Clone this repo
2. cd path and `npm install`
3. `cd reactjs` and `npm install`
4. `cd..` and `npm start`
5. new terminal, cd to reactjs and `npm start`
